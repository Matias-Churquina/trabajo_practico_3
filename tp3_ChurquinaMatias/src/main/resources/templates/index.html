Teoría

A continuación se presetará los contenidos y enlaces al material teórico presentados semana a semana vistos hasta el momento.

Semana 1

En esta semana se aborda la unidad teórica referida a temas como la comprensión y manipulacón de la plataforma JAVA. También temas referidos a la programación JAVA (reglas sintáctica y semántica). Intorducción a JAVA Programación JAVA

Semana 2

Se presentan temas referidos a atributos, contructores, y métodos accesores. Se introduce tipo de dato String y Fechas, tambén se describe el mecanismo de conversión de tipos de datos, y documentación de código.

Sobrecarga String y variables miembro Conversión de tipos de Datos Fechas en java Documentación de código

Semana 3

Aquí se presenta la introducción a los proyectos Maven, y se profundiza el versionamiento con Git. Por otra parte se brinda una breve conceptualiazación al uso de interfaces.

Proyectos Maven Interfaces

Semana 4 y 5

Se describe el uso de enumerados, uso de las funcionalidades que ofrece JAVA para la programación funcional.

Programación funcional Enumerados

Semana 6

En esta semana se conceptualiza la arquitectura de aplicaciones web y técnologias usadas, framework, spring framework, spring boot, y se realiza una introducción a HTML.

Aplicaciones web Arquitectura de Aplicaciones Web Framework-Spring Framework HTML 5
